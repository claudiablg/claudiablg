## Quickstart

### Prerequisites

- bash terminal (MacOs, or WSL 2 on Windows 10)
- [nvm](https://github.com/nvm-sh/nvm)
- [brew](https://brew.sh/)

### Get Started

```bash
# set permissions for scripts
chmod 755 scripts

# install correct version of node
yarn nvm

# install dependencies
yarn

# build packages
yarn build

# test packages
yarn test

# apply prettier and eslint
yarn lint

```
